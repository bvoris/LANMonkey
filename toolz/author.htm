ABOUT LANMonkey
LANMonkey was originally a DOS based scripting menu I (Brad P. Voris) developed for network diagnostic. It was designed to make day to day network tasks easier and less time consuming. My original script project known as IPToolz is still in use and is updated as network protocols change inside of Microsoft Windows desktop and server environment.
Currently LANMonkey Supports Microsoft Windows 2000, XP Professional and Home
